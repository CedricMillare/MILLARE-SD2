Log In And Log Out

This project is a simple login and logout system developed using Python in PyCharm, with the Tkinter library for the graphical user interface (GUI).

The system allows users to enter a username and password through a login window. If the entered credentials match the stored data, the user is successfully logged into the system and redirected to the main interface. The interface also includes a logout button that allows the user to safely exit the session and return to the login screen. The project demonstrates basic concepts of GUI design, user authentication, and event handling in Python. Tkinter is used to create windows, labels, text fields, and buttons, making the application interactive and easy to use. This system can serve as a beginner-level example for understanding how login systems work in desktop applications and can be expanded in the future to include features such as user registration, password validation, and database integration.
